📋 Overview

The Verity-JE-Mod-Minecraft adds new content to your Java Edition game. This project focuses on custom progression and unique mechanics for your Minecraft world. It works on versions 1.8 and 1.16.5. You can use it in standard worlds, skyblock maps, or modpacks.

💻 System Requirements

Verify your computer meets these needs before you start:

  • Windows 11 or Windows 10
  • Minecraft Java Edition installed
  • Java Runtime Environment (JRE) compatible with your Minecraft version
  • At least 4GB of RAM allocated to Minecraft

⚙️ Preparing for Installation

Minecraft requires a mod loader to run custom additions. This mod supports Forge. Download the correct version of Forge that matches your Minecraft version. Run the Forge installer and click "Install Client." After the process finishes, launch the Minecraft Launcher once to verify the Forge profile appears.

📥 How to Download and Install

Follow these steps to add the mod to your game:

  1. Visit the official repository page to access the files.
  2. Locate the "Releases" section on the right side of the page.
  3. Click the latest version link to download the .jar file to your computer.
  4. Press the Windows Key and R on your keyboard at the same time.
  5. Type %appdata% into the box and press Enter.
  6. Open the .minecraft folder.
  7. Locate the "mods" folder. If you do not see one, create a new folder and name it "mods" exactly.
  8. Drag and drop the downloaded .jar file into this folder.

🎮 Playing the Mod

Open your Minecraft Launcher. Select your Forge profile from the menu at the bottom left. Click Play. On the main menu, click the Mods button to ensure Verity-JE-Mod-Minecraft appears in your list. Create a new world to test the new features.

🔧 Troubleshooting

If the game crashes, check your version of Minecraft. The mod version must match your version of Minecraft. Remove other mods if you experience compatibility issues. Ensure you have the correct Java version installed. If you use a custom launcher, make sure you placed the file in the correct instance directory.

Does this work on Bedrock edition? No, this is for Java Edition only.

Can I use this on a server? Yes, install the mod file to your server mod folder and your client mod folder to play with friends.

Will it break my current world? Always back up your worlds before you add new mods.
